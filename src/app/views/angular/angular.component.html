Configurando o Ambiênte de Desenvolvimento!

Instale o NODE.JS em sua máquina para ter acesso aos pacotes NPM, como o "npm install" que é utilizado para a instalação do Angular/cli.
Depois de instalado, em seu prompet de comando escreva "node --version" para verificar se foi instalado corretamente.

Na sequencia instale o Angular/cli! Você pode instalar na ultima versão ou em alguma especifica, como @10.2.3.

{{ installAngular }}

{{ installAngularVersion }}

Se for preciso desinstalar, utilize os comandos.

{{ uninstallAngular }}

{{ cleanCache }}

Instale o VS-CODE para começar a desenvolver suas aplicações!
